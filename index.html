Guide

Two ways to install

Use the desktop app for the fastest route, or use Charon Gen when you want a browser-generated ZIP package.

01

Method 1 - Automatic Install

  1. Download Charon from the download button.
  2. Open Charon and let it detect your Steam folder.
  3. Search or generate from the App ID.
  4. Install automatically and let Charon prepare the required files.
  5. Launch Steam and start from the library.
02

Method 2 - Manual Install

  1. Open Charon Gen from the navigation.
  2. Enter the Steam App ID and wait for game details.
  3. Generate or resolve a ZIP package.
  4. Download the ZIP and keep it ready.
  5. Open Charon and drag the manifest ZIP into the manual drop area.
  6. Let Charon place the Lua and manifest files into the correct Steam folders.

FAQ

Questions before downloading?

Short answers for the things people usually ask before trying Charon.

Is Charon portable?

Yes. Charon is shared as a portable Windows `.exe`, so users can run it without a traditional installer.

Where does Charon place Lua and manifest files?

Lua files go into Steam's `config/stplug-in` folder, and `.manifest` files go into Steam's `depotcache` folder.

Does it work with every game?

Charon depends on available Lua and manifest packages. If a package is missing or broken, use the Discord community to request or report it.

What is the automatic install limit?

The desktop app currently allows 10 automatic manifest installs per 24 hours.

Can I install a ZIP manually?

Yes. Download or generate a ZIP, then drag it into the manual import area inside Charon.
